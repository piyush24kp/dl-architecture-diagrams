LoRA
Low-rank adapters only
Freeze W, train B·A matrices
0.1-1% of params, same quality
ΔW = B·A r=4–64
✅

fits 70B model
with LoRA +
gradient ckpt
04
QLoRA
Quantize base + LoRA on top
Base: 4-bit NF4 quantization
Adapters: full precision (bf16)
70B model fits on 2× A100 40GB
✅
Dettmers 2023
Democratised
LLM fine-tuning
on consumer GPU

✅

large scale (>11B)
