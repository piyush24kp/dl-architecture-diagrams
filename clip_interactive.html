05
Contrastive Loss
InfoNCE / NT-Xent Loss
Maximize diagonal, minimize rest
Symmetric: image→text + text→image
L = -log(e^(sᵢᵢ/τ) / Σⱼ e^(sᵢⱼ/τ))
✅
Batch size is
crucial! Larger =
more negatives
