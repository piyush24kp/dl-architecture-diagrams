01
Vanilla RNN
Hidden state carries memory
hₜ = tanh(W·[hₜ₋₁, xₜ] + b)
Same weights at every timestep
hₜ = tanh(W_h·hₜ₋₁ + W_x·xₜ)
✅
Processes one
token at a time
Cannot parallelise
→ slow training
02
Vanishing Gradient
Critical failure of vanilla RNN
Gradient shrinks through BPTT steps
"The cat … was hungry" — fails at 20+ steps
∂L/∂h₀ ≈ 0 after t steps
✅
Exploding grad
also occurs:
gradient clipping
patches it
03
LSTM Gates
3 gates control memory flow
Forget gate: what to erase from cell
Input gate: what new info to add
Output gate: what to expose
cₜ = f⊙cₜ₋₁ + i⊙g̃ (cell state)
✅
Hochreiter &
Schmidhuber
1997 — solved
vanishing grad
04
LSTM Cell State
Highway for long-range memory
Cell state flows almost unchanged
Gates add/remove information
cₜ = forget⊙cₜ₋₁ + input⊙g̃
✅
Cell state is
the key insight
allows 1000+
step memory
05
GRU Simplification
Merge cell + hidden state
Only 2 gates: Reset + Update
Fewer params, similar performance
hₜ = (1-z)⊙hₜ₋₁ + z⊙h̃ₜ
✅
Cho et al. 2014
~33% fewer
params than
LSTM
06
Bi-directional RNN

Each token sees left AND right context
Used in early NLP (ELMo, pre-BERT)
hₜ = [h→ₜ ; h←ₜ] (concatenate)
✅
Can't be used
for generation
(needs future)
only encoding
